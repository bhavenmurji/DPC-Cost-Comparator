1. **Day 0**: Template 1 (Initial outreach)

3. **Day 14**: Template 4 (Re-engagement with urgency)

## Personalization Variables

Use these placeholders when importing to email tool:

- `[Provider Name]` - Dr. Smith, Jane, etc.
- `[Practice Name]` - Smith Family Medicine
- `[City]` - Los Angeles, New York, etc.
- `[State]` - CA, NY, etc.
- `[ZIP code]` - 90210, 10001, etc.
- `[Month]` - January, February, etc.
- `[X]` - Dynamic numbers (views, clicks, searches)
- `[Current Monthly Fee]` - $75, $100, etc.
- `[Platform URL]` - ignitehealthpartnerships.com
- `[Your Name]` - Your actual name
- `[Your Email]` - Your contact email
- `[Phone]` - Your phone number

## Email Tool Recommendations

### For Notion
1. Create a database with columns: Template Name, Subject, Body, Sequence, Day, Status
2. Add filter views: "Active Templates", "By Sequence", "By Provider Stage"
3. Use linked databases to track which providers received which emails

### For Airtable
1. Base: "Provider Communications"
2. Tables:
   - Email Templates (this content)
   - Provider Contact List
   - Email Sends (tracking)
   - Performance Metrics
3. Automations: Send monthly reports, follow-up sequences
4. Views: By template type, by send status, by provider segment

### For SendGrid/Mailchimp
1. Import templates as stored templates
2. Create segments: New Providers, Claimed Providers, High-Volume Providers
3. Set up automated sequences based on provider actions
4. Track: Open rate, click rate, response rate, claim rate
